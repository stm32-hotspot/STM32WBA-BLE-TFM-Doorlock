Purpose

This package contains a specific application to demonstrate how to create a Bluetooth Low Energy Secure Application on STM32WBA in a TrustZoned environment running the TF-M, see README.md file for more details

BLE-TFM-Doorlock example is being delivered as-is, not necessarily supported by ST.

.

This example is based on the STM32CubeWBA_v1.1.0 package. For further information about these components, see official package.
